# Prova-Senac


O intuito é de fazer um sistema que possa ser realizado o CRUD de produtos com administração de usuarios/permissões

# Como rodar:

Backend: 
Entre na pasta do backend -> cd backend
E depois na pasta de Rotas -> cd Routes
E então rode o php local -> php -S localhost:8000

Frontend: 
Entre na pasta do frontend -> cd frontend
Instale se necessário as dependências -> npm i
E então inicie o vuetify -> npm run dev



# Ferramentas/Linguagens/Frameworks utilizados:

Backend: 
PHP puro

Frontend:
Vuetify (com vue)
